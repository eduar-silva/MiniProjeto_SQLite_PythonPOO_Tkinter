# MiniProjeto
Mini Projeto usando SQLite3, Python orientado a objeto e a biblioteca tkinter para criação de uma simples interface, onde são adicionados: nome, sobrenome, email e CPF. A informação é adicionada ao banco de dados e apresentada em uma telinha ao lado. <img width="970" height="373" alt="image" src="https://github.com/user-attachments/assets/7d18d612-a1d6-4426-97cb-8b1f793e3dc0" />
